Request Header Parser Microservice API

Coded by Joshua Davidson

Description

A simple request header parser API that returns a JSON containing your IP address, language, and software.

Visit https://request-header-parser-project.herokuapp.com/parseheader to give it a try!

Example Output

{ ipaddress: "8.8.8.8", language: "en-US", software: "Windows NT 10.0; WOW64" }
